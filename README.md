# Knowzy

## 📌 Descrição
Aplicativo interativo onde estudantes possam testar seus conhecimentos de forma rápida, prática e organizada. A plataforma será dividida por matérias como Matemática, Português e História, permitindo que o usuário escolha o conteúdo que deseja praticar.

## 🎯 Objetivo
Interface intuitiva aonde o usuário pode escolher quizzes separados por matérias, favoritar um quiz específico, exibir desempenho e conquistas do usuário.


## 🧩 Tecnologias Utilizadas

### Backend (2TW)
- Node.js
- Firebase

### Mobile
- React Native

## 🚀 Funcionalidades
- Cadastro e login
- Escolha de matérias
- Sistemas de quizzes
- Pontuação
- Temporizador de perguntas
- Página de perfil
